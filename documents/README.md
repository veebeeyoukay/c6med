# C6Med Documents Directory

## Overview

This directory contains C6Med's comprehensive cybersecurity documentation, policies, and procedures. All documents are organized by category for easy navigation and reference.

---

## Document Structure

- **0. Plan/**
  - Security tracking and project planning documents
  
- **1. Risk Assessment/**
  - Cybersecurity assessment reports and risk registers
  
- **2. WISP/** (Written Information Security Program)
  - Information security policies and procedures
  
- **3. Training/**
  - Cybersecurity and technical training materials
  
- **4. Insurance/**
  - Insurance application support documents
  
- **5. Vulnerability Management/**
  - Vulnerability and patch management procedures
  
- **7. Standard Operating Protocols/**
  - Technical SOPs and training protocols
