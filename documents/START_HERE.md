# Start Here

## File Explorer Root Directory

**This folder (`/documents/`) is the root directory for the file explorer in the dashboard.**

All files placed in this folder and its subfolders will be accessible through the dashboard file explorer.

---

## Folder Structure

- **Medical Communications/**
  - Case Studies/
  - Presentations/
- **Client Documents/**
  - Pharma Co A/
- **Resources/**
  - Templates/

## Adding Files

Place your files in the appropriate folders. The file explorer will automatically display them in the dashboard.

Supported file types:
- PDF documents (.pdf)
- Word documents (.docx, .doc)
- PowerPoint presentations (.pptx, .ppt)
- Text files (.txt)
- Images (.png, .jpg, .jpeg)
